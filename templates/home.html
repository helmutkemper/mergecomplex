Bem-vindo ao Monaco Editor Site

Este projeto demonstra a integração do poderoso Monaco Editor (o mesmo editor usado no Visual Studio Code) com um servidor web em Go.

Características do Projeto

Monaco Editor

Editor de código completo com syntax highlighting, autocomplete e temas personalizáveis.

Servidor Go

Backend leve e performático usando a linguagem Go para servir o aplicativo.

Menu Intuitivo

Navegação lateral com ícones Font Awesome para fácil acesso às funcionalidades.

Open Source

Projeto totalmente open source, utilizando apenas tecnologias livres e gratuitas.

Tecnologias Utilizadas

  • Go (Golang) - Linguagem de programação para o backend
  • Monaco Editor - Editor de código da Microsoft
  • Font Awesome Free - Biblioteca de ícones
  • HTML5 & CSS3 - Estrutura e estilo da interface

Como Começar

Navegue pelo menu lateral para:

  1. Editor - Experimente o Monaco Editor em ação
  2. Diff Editor - Compare dois arquivos lado a lado
  3. Sobre - Saiba mais sobre o projeto e suas funcionalidades
