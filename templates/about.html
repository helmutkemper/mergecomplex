Sobre o Projeto

O que é este projeto?

Este é um projeto de demonstração que integra o Monaco Editor da Microsoft com um servidor web escrito em Go (Golang). O objetivo é mostrar como criar uma aplicação web moderna e performática utilizando apenas tecnologias open source.

Monaco Editor

O Monaco Editor é o editor de código que alimenta o Visual Studio Code. Ele oferece recursos avançados como:

  • Syntax highlighting para múltiplas linguagens
  • IntelliSense e autocomplete
  • Validação de código em tempo real
  • Suporte a temas personalizáveis
  • Diff editor para comparação de código lado a lado
  • Busca e substituição avançada
  • Suporte a múltiplos cursores

Repositório oficial: github.com/Microsoft/monaco-editor

Por que Go?

Go foi escolhido para o backend por diversos motivos:

  • Performance: Compilado e altamente eficiente
  • Simplicidade: Sintaxe limpa e fácil de aprender
  • Concorrência: Goroutines para processamento paralelo
  • Biblioteca padrão: HTTP server embutido, sem dependências externas
  • Deploy simples: Binário único, sem runtime necessário

Como Executar

  1. Certifique-se de ter o Go instalado (versão 1.16+)
  2. Clone ou baixe o projeto
  3. Execute o comando: 
    go run main.go
  4. Acesse http://localhost:8080 no navegador
